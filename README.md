## 🧠 Mission Brief

**CyberShell Academy** is a terminal-based cyber training RPG that teaches Windows, Linux, PowerShell, GitHub, and defensive cyber fundamentals through missions, XP, ranks, daily challenges, boss trials, badges, and local operator profiles.

This project exists for one reason:

> **Turn command-line practice into a playable cyber training experience.**

No boring wall of theory. No random copy-paste commands with no context. This repo is built to help people practice, break through fear of the terminal, and build real technical confidence one mission at a time.

---

## 🚀 What Makes This Different

Most beginner cyber projects are either too basic, too dry, or too advanced too fast. **CyberShell Academy** is built like a training pipeline.

You do not just read about commands.

You run them.  
You answer challenges.  
You earn XP.  
You unlock missions.  
You fight boss trials.  
You build command-line muscle memory.

This repo is designed for:

- new cybersecurity students
- Linux beginners
- PowerShell learners
- homelab builders
- future system administrators
- ethical hacking students
- anyone tired of passive tutorials
